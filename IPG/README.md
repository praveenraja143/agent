# IPG - LinkedIn Auto Poster

## Features
- Resume upload and auto skill extraction
- Certificate upload with AI content generation
- Auto-post to LinkedIn with smart hashtags
- Auto-update resume with new certificate skills
- Daily auto-post scheduling (3x per day)
- Job search with skill matching
- Post history tracking
- WhatsApp notifications

## Setup
```bash
cd IPG
npm install
npx expo start
```

## How to Use
1. Upload your resume (PDF/DOCX)
2. Enter LinkedIn credentials
3. Get free OpenRouter API key from openrouter.ai
4. Upload certificates and auto-post to LinkedIn
5. Resume auto-updates with new skills from certificates
